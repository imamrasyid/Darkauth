# Examples Folder Planning

Ready-to-run examples to help developers understand how to use the library.

## Planned Examples:
- `ci3_integration.php`: How to load Darkauth as a CI3 library.
- `basic_login.php`: Simple session-based login script.
- `api_jwt.php`: Example of issuing and validating JWTs.
- `custom_driver.php`: How to implement a custom authentication driver.
