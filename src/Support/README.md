# Support Folder Planning

Helper classes and utilities that don't belong to a specific architectural layer.

## Planned Utilities:
- `Hash.php`: Wrapper for PHP's `password_hash` and `password_verify`.
- `Config.php`: Simple configuration loader.
- `Collection.php`: (Optional) A simple wrapper for arrays to make data manipulation easier.
- `JwtHelper.php`: Wrapper for `firebase/php-jwt` to simplify token generation and validation.
