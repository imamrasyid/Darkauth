# Tests Folder Planning

Comprehensive testing using PHPUnit.

## Structure:
- `Unit/`: Testing individual classes in isolation.
- `Integration/`: Testing multiple components working together (e.g., AuthManager + SessionGuard).
- `Mocks/`: Mock objects for Users and Databases.

## Coverage Goals:
- High coverage for Core and Drivers.
- Ensure compatibility across PHP 7.4 to 8.x.
